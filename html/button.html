Buttons

다양한 크기, 상태 등을 지원하는 폼, 대화 상자 등의 작업에 부트 스트랩의 사용자 정의 단추 스타일을 사용하십시오.

Native

얘네들은 영향을 받으면 안 됨

a link 
<input type="button" value="input button">
<input type="submit" value="input submit">
<input type="reset" value="input reset">
<button type="button">button button</button>
<button type="submit">button submit</button>
<button type="reset">button reset</button>
<a href="#link">a link</a>

기본형

버튼 기본형태

<button class="xu-button xu-button--default">(default)</button>
<button class="xu-button xu-button--primary">primary</button>
<button class="xu-button xu-button--link">link</button>
<button class="xu-button xu-button--subtle">subtle</button>
<button class="xu-button xu-button--subtle-link">subtle-link</button>
<button class="xu-button xu-button--warning">warning</button>
<button class="xu-button xu-button--danger">danger</button>
<button class="xu-button xu-button--default xu-button--selected">selected all</button>

Size

버튼 사이즈

<button class="xu-button xu-button--primary">default</button>
<button class="xu-button xu-button--primary xu-button--small">small</button>
<button class="xu-button xu-button--primary xu-button--large">large</button>

Block

블럭 형태

<button class="xu-button xu-button--default xu-button--block">Block level button</button>
<button class="xu-button xu-button--primary xu-button--block">Block level button</button>

Active

active 상태, focus 상태, hover 상태

<button class="xu-button xu-button--primary active">active</button>
<button class="xu-button xu-button--primary focus">focus</button>
<button class="xu-button xu-button--primary hover">hover</button>

Disabled

비활성화 상태, 클래스와 attr 의 disabled 속성을 같이 적용

<button class="xu-button xu-button--default disabled" disabled>disabled</button>
<button class="xu-button xu-button--primary disabled" disabled>disabled</button>

비활성화 상태, a 태그는 disabled 가 없음 클래스를 붙여서 적용

default primary 
<button class="xu-button xu-button--default disabled" tabindex="-1" role="button" aria-disabled="true">disabled</button>
<button class="xu-button xu-button--primary disabled" tabindex="-1" role="button" aria-disabled="true">primary</button>

Icon

아이콘 버튼
class="xu-button__icon" 영역에 class="xu-button__icon--pen" 에 아이콘 이미지가 적용되어 있습니다.
커스텀된 아이콘을 사용하고 싶다면 class="xu-button__icon--pen" 대신 아이콘 이미지를 적용한 클래스를 적용하면 다른 아이콘들도 적용할 수 있습니다. 

<button class="xu-button xu-button--default xu-button--icon">
<span class="xu-button__icon xu-button__icon--pen"></span>
</button>

</button>
